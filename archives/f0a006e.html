在CentOS 8安装Rstudio Server

由于处理数据太大导致本地电脑内存不够，因此才想着利用服务器安装Rstudio server。服务器版本的Rstudio可以通过网页版本访问：IP地址+端口。

PS： 安装系统是CentOS8, Rstudio Server的安装需要管理员root权限。

安装记录

下载

PS：不同版本的系统，下载Rstudio server软件和方式都不一样

  • 安装R，Rstudio server依赖R
  • 下载Rstudio server

安装

必须在root或拥有root权限的账户下安装

  • 创建Rstudio配置文件，并将如下两行内容添加进入文件
  • rsession-which-r=/disk/share/anaconda3/bin/R : 指定R路径
  • www-port=8787: 指定网页访问端口
  • 打开8787端口

centos 8

ubuntu 20

启动

出现Active: active (running) since，则安装成功，可通过本地浏览器服务器IP地址:8787打开
例如：192.168.106.31:8787。账户为Linux服务器账户。

用户管理

安装过程遇到的bug

1 运行sudo rstudio-server verify-installation时出现

删除/etc/rstudio/rsession.conf，重启 sudo systemctl start rstudio-server.service。

2 登陆过程Rstudio initialization error unable to connect to service，这是因为SELinux（Security-Enhanced Linux）设置的问题

  • 报错信息: 是因为SELinux阻止了外界访问

  • 处理方法1，安装报错信息提示进行

  • 处理办法2，关闭SELinux（Work）

3 在ubuntu20开启防火墙后，导致无法使用ssh命令登陆服务器ssh: connect to host 192.168.106.31 port 22: Connection timed out，使用如下命令关闭防火墙即可:
