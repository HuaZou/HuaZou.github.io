机器学习（一）：初步认识机器学习

Posted on 2021-01-02 In Course , Machine learning Views: Valine:
Symbols count in article: 7.3k Reading time ≈ 7 mins.

介绍

深知自己在机器学习上的不足，本文只是基于应用层面（R语言的caret包）记录自己如何入门机器学习实操，归纳总结学习过程的心得体会。根据 Dr. Jason Brownlee的教程入门机器学习。该教程主要包括：

  1. 加载数据

  2. 汇总数据（描述性统计量）

  3. 可视化数据

  4. 评估不同模型的算法

  5. 基于模型预测

加载数据

数据集分割

评估模型优劣时候，一般会对数据集进行划分：训练集和测试集。在使用训练集构建模型后，可使用模型对测试集进行预测分析，最后根据ROC下的AUC对模型进行评估。

数据划分有多种方式，既有分层随机抽取，也有根据概率划分的方式

汇总数据

汇总数据的目的是为了评估数据，一般会做如下操作

  • 数据维度
  • 数据属性
  • 部分数据
  • 分类变量因子
  • 分类数据量
  • 描述统计量

可视化数据

前面描述性统计量是我们对数据有了大概的了解，但还是不够直观，这个时候可视化数据就能更好帮助我们深入了解数据。

  • 箱型图：展示单个变量数据分别情况

  • 散点图：不同变量在分组间的分布情况

  • 分组箱线图: 同一变量不同分组的区别

  • 密度曲线图：同一变量在不同分组的密度分布

评估不同模型的算法

选择算法对训练集进行构建模型，再基于测试集评估模型优劣。一般做如下处理：

  1. 设置测试参数；

  2. 基于算法构建模型；

  3. 选择最佳模型。

  • 设置参数：10倍交叉，3次重复十倍交叉等方法和评估参数accuracy以及其他参数等。 10倍交叉验证会将数据分成10分，9份训练和1份测试
  • 构建模型：选择不同的算法构建不同的模型。选择bagging(RF)和Stacking(LDA/CART/kNN/SVM)两类算法。

  1. Linear Discriminant Analysis (LDA)：简单线性方法

  2. Classification and Regression Trees (CART)：非线性方法

  3. k-Nearest Neighbors (kNN)：非线性方法

  4. Support Vector Machines (SVM) with a linear kernel：复杂非线性方法

  5. Random Forest (RF)：复杂非线性方法

选择最佳模型

评估模型优劣，可以通过Accuracy指数（预测正确样本数比上总体样本数的百分值）

也可以通过可视化展示结果

也可单独展示某个算法的结果

基于模型预测

从比较结果得知LDA/kNN算法的模型是最佳模型，那么它们在测试集也就是验证集的效能如何呢。
