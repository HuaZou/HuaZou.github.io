机器学习（七）：集成学习算法

Posted on 2021-01-16 In Course , Machine learning Views: Valine:
Symbols count in article: 9.3k Reading time ≈ 8 mins.

介绍

通常我们会花费较多时间寻找适合数据集的模型算法，不同算法的模型会得到不同的模型效能，三种常用的算法类型：

  • Bagging: 从训练集进行子抽样组成每个基模型所需的子训练集，对所有基模型预测的结果进行综合产生最终的预测结果（Random Forest 随机森林）

  • Boosting: 训练过程为阶梯状，基模型按次序—进行训练（可并行实现），基模型的训练集按照某种策略每次都进行一定的转化，对所有的基模型预测结果进行线性综合产生最终的预测结果。 （随机梯度算法 Stochastic Gradient Boosting）

  • Stacking: 将训练好的所有基模型对训练基进行预测，第j个基模型对第i个训练样本的预测值将作为新的训练集中第i个样本的第j个特征值，最后基于新的训练集进行训练。同理，预测的过程也要先经过所有基模型的预测形成新的测试集，最后再对测试集进行预测

Dataset

Bagging Algorithms

Bagging Algorithms又称装袋算法，从数据集中随机抽取m个样本，重复n次生成n个训练集，然后训练模型得到n个分类器，最后根据分类器投屏决定输出结果。常有的算法有Bagged CART和Random Forest。

Bootstrap aggregating通常又简称为Bagging(装袋法)，它是让各个模型都平等投票来决定最终结果。为了提高模型的方差(variance, 差异性)，bagging在训练待组合的各个模型的时候是从训练集合中随机的抽取数据

Boosting Algorithms

Boosting Algorithms的Boosting是一种框架算法,主要是通过对样本集的操作获得样本子集,然后用弱分类算法在样本子集上训练生成一系列的基分类器。常用算法有C5.0和Stochastic Gradient Boosting（随机梯度提升算法）。

Boosting(提升法)是通过不断的建立新模型而新模型更强调上一个模型中被错误分类的样本，再将这些模型组合起来的方法。在一些例子中，boosting要比bagging有更好的准确率，但是也更容易过拟合。

Stacking Algorithms

Stacking(有时候也称之为stacked generalization)是指训练一个模型用于组合(combine)其他各个模型。即首先我们先训练多个不同的模型，然后再以之前训练的各个模型的输出为输入来训练一个模型，以得到一个最终的输出。如果可以选用任意一个组合算法，那么理论上，Stacking可以表示上面提到的各种Ensemble方法。然而，实际中，我们通常使用单层logistic回归作为组合模型。

使用五类算法做stacking算法：

  • Linear Discriminate Analysis (LDA)

  • Classification and Regression Trees (CART)

  • Logistic Regression (via Generalized Linear Model or GLM)

  • k-Nearest Neighbors (kNN)

  • Support Vector Machine with a Radial Basis Kernel Function (SVM)

Notes: 针对同一个数据集，不同算法的模型效能

评估不同算法模型的效能相关性

综合五类算法模型后再用线性模型训练数据：相比单独使用某一类算法，集成学习模型能提高效能

总结

集成机器学习算法的方法根据其训练模型方式不同，有三类：Bagging，Boosting和Stacking。
