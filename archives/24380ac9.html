线性回归

线性回归是使用最广泛且最简单的统计预测模型方法。它的数学公式：

$Y$是 dependent variable, $X$是independent variable，$\theta$是coefficient。系数反映了某些independent variable对dependent variable的重要性

线性回归的最终目的是：模型预测值与真实值差值越来越小，差值通常用残差（Sum of residuals）表征:

$h(x)$ 是预测值，$y$是真实值，残差和也叫做均方误差（Mean Squared Error: MSE）

岭回归和lasso回归

测试数据在训练模型表现出较差的预测效果，这是因为模型发生了过拟合的结果。岭回归和lasso回归是通过引入少量偏差纠正过拟合的现象，对损失函数施加惩罚项降低过拟合的风险，并且降低自变量间的多重共线性带来的信息冗余。

岭回归（Ridge Regression）的损失函数：

相比线性回归的损失函数，它添加了以$\lambda$作为系数的误差项作为惩罚函数，这也是正则化。

重点：

  • It shrinks the parameters, therefore it is mostly used to prevent multicollinearity.
  • It reduces the model complexity by coefficient shrinkage.
  • It uses L2 regularization technique.

lasso回归(Least Absolute Shrinkage Selector Operator)的损失函数：

对比岭回归的惩罚项，lasso回归是$\theta$的绝对值，而岭回归是平方和。

重点：

  • It uses L1 regularization technique.
  • It is generally used when we have more number of features, because it automatically does feature selection.

两者的异同点：

  • 岭回归和lasso回归均是通过添加惩罚项降低过拟合现象；
  • 岭回归的惩罚项中$\theta$是平方和也即是L2正则，而lasso回归的$\theta$是绝对值也即是L1 regulation;
  • 岭回归只会减小自变量的回归系数，而lasso回归可以将回归系数降至0；
  • 对大量features进行筛选，建议使用lasso回归，但当存在线性相关的变量时，lasso回归只会选择其中之一的变量，这可能会导致丢失某些有用信息最后导致模型精确度下降，通常大家会使用弹性网络回归(Elastic Net Regression);
  • 岭回归和lasso回归是不同正则化的结果，当regulation是L1则是岭回归，是L2则是lasso回归。

弹性网络回归

弹性网络回归是结合了L1和L2两种正则化方式对损失函数进行处理的回归方法，因此我们可以选择Lasso或者岭回归方法去处理数据，其损失函数：

原理是：先将存在相关的自变量归为一组，如果该组的某个自变量与因变量存在强相关关系，则将整个组纳入模型构建中来。假如$a$和$b$分别是L1和L2的权重:

假定L1_ratio = a /(a+b) 且 (a+b) = 1

  • If l1_ratio =1, therefore if we look at the formula of l1_ratio, we can see that l1_ratio can only be equal to 1 if a=1, which implies b=0. Therefore, it will be a lasso penalty.
  • Similarly if l1_ratio = 0, implies a=0. Then the penalty will be a ridge penalty.
  • For l1_ratio between 0 and 1, the penalty is the combination of ridge and lasso.

模型求解

在求解MSE的过程中，通常使用损失函数（The cost Function）计算模型的MSE：

cost function和误差平方和类似，仅仅在前面加了$\frac{1}{2m}$系数。

求解损失函数最小值的方法有两类：1.最小二乘法(least square method)；2.梯度下降法(Gradient Descent)

  1. 最小二乘法：找到一组$\theta{n}$是的损失函数$J(\theta{0}, \theta_{1})$的残差平方和最小，对损失函数求导，并取值为0即是的损失函数具有最小值。

  2. 梯度下降法：对损失函数的$\theta$求偏导，找到损失函数下降最快的方法。

  1. 给定待优化连续可微函数$J\left ( \Theta \right)$、学习率$\alpha$以及一组初始值 $\Theta{0}=\left ( \theta{01}, \theta{02}, \cdots, \theta{0l}, \right )$；
  2. 计算待优化函数梯度：$\triangledown J\left ( \Theta _{0} \right )$；
  3. 更新迭代公式：$\Theta^{0+1}=\Theta {0}-\alpha \triangledown J\left ( \Theta {0} \right )$；
  4. 计算$\Theta^{0+1}$处函数梯度$\triangledown J\left ( \Theta _{0+1} \right)$；
  5. 计算梯度向量的模来判断算法是否收敛：$\left| \triangledown J\left ( \Theta \right ) \right |\leqslant \varepsilon$；
  6. 若收敛，算法停止，否则根据迭代公式继续迭代。

这时，梯度下降法是一个不错的求解策略。

评估模型指标

  • 误差平方和（Mean Square of Error）：该值越小则模型拟合越好，但可能存在过拟合现象，并且其值的大小和样本数量有密切关系(预测值和真实值的差值平方)：

  • 最好衡量线性回归法的指标：R square

    ​

$\sum_{i=1}(\bar{y}^i-y^i)^2$是基准模型，使用$y=\bar{y}$预测产生的错误，可理解为所有预测样本的预测结果都是样本均值。分子是模型预测产生的错误，分母是基准模型产生的错误，那么商值就是模型产生错误的比率，也即是拟合错误的比率，从而推测出$R^2$即是拟合成功的比率。
$R^2$取值范围是[0, 1], 其值越大则拟合效果越好，但当$R^2$ < 0, 则模型的自变量和因变量可能不存在任何相关关系。

  • 预测准确率，可以用AUC（混淆矩阵）等评估分类器效能。

  • Adjusted R square是对新添加自变量的$R^2$进行校正，其取值的变化可以反映模型拟合效果（有时候添加变量可能会导致$R^2$增大，但实际上这种增加是虚假的）：
