生存分析基础

Posted on 2020-04-14 In R , DataAnalysis Views: Valine:
Symbols count in article: 9.2k Reading time ≈ 8 mins.

介绍

​ 生存分析是利用如Kaplan-Meier方法和Log-Rank检验评估某段时间内某件事的生存概率和风险概率，从而发现某些因素对生存的影响如何。生存分析适用于多个领域：

  • 癌症研究中的病人存活时间分析；

  • 社会学上的”event-history analysis“；

  • 工程管理学里面的”failure-time analysis“.

在癌症研究中，典型的研究问题如下：

  • 哪一项临床指标对病人的生存有显著地影响；
  • 某位病人存活三年的概率有多少；
  • 两组病人的生存是否存在显著差异呢？

目的

癌症研究中，生存分析常用到的方法是如下几类：

  • Kaplan-Meier plots可视化生存曲线，该方法先计算生存概率后可视化该结果；
  • Log-rank test 比较组间的生存曲线，该方法会计算得到p值能反应组间的生存曲线是否显著差异；
  • Cox proportional hazards regression 计算变量对生存事件的效应，其主要使用了Cox regression模型。

生存分析需要了解的基本分析原则：

  • 如何生成和解释生存曲线；
  • 如何定量且检验病人组间的生存差异；

基本概念

生存分析的基本概念：

  • 生存时间和事件；
  • Censoring(终检)；
  • 生存函数和风险函数；

癌症研究中的生存事件和事件类型

  • Relapse（复发）

  • Progression（连续）

  • Death（死亡）

    The time from ‘response to treatment’ (complete remission) to the occurrence of the event of interest is commonly called survival time (or time to event).

癌症研究中最重要的两类指标：

  • 死亡时间the time to death;

  • 无复发生存时间 the relapse-free survival time :

    the relapse-free survival time, which corresponds to the time between response to treatment and recurrence of the disease. It’s also known as disease-free survival time and event-free survival time.

Censoring

生存分析关心某个时间如死亡和复发的发生时间。然后这些事件可能因为其他因素导致无法观察，这样的数据就认为是censored observations。这样的观察值发生的情况如下：

  • 病人在整个研究期间均没有发生研究者希望的事件；
  • 病人在研究期间无法追踪下去；
  • 病人发生其他不同的意外事件导致无法适合进一步追踪。

这样的”censored observations“被称为right censoring，可以在生存分析中处理。

Survial and hazard functions

描述生存数据的两类概率是：生存概率survival probability和风险概率hazard probability。

  • 生存概率函数$S(t)$: 它计算个体最初的时候到某个特定的未来时期的生存概率；
  • 风险概率函数$h(t)$: 个体在某个时间段的某个时间点处于某种观察值状态下的概率；

Note that, in contrast to the survivor function, which focuses on not having an event, the hazard function focuses on the event occurring.

Kaplan-Meier survival estimate

Kaplan-Meier方法是一个非参数方法，其被用来评估在观察的生存时间下的生存概率，是由Kaplan 和 Meier在1958年提出，具体公式如下：$S(t_{i})$

  • $S(t{i})$ = the probability of being alive at t{i-1}$;
  • $n_i$= the number of patients alive just before $t_i$;
  • $d_i$ = the number of events at$t_i$;
  • $t_0$; = 0, $S(0)$ = 1;

$S(t)$函数仅仅与事件的时间有关，随着时间变化，事件的数目也发生变化。KM 曲线概率与时间成负相关，可以计算概率的置信区间。

R中的生存分析

安装R包

主要安装两个R包：

  • survival：生存分析计算；

  • surviminer： 归纳总结和可视化生存分析结果；

案例数据

survival包自带的肺癌数据：

  • inst: Institution code # 可理解为病人ID
  • time: Survival time in days # 生存时间
  • status: censoring status 1=censored, 2=dead # 事件状态，1终止；2死亡
  • age: Age in years # 病人年龄
  • sex: Male=1 Female=2 # 病人性别
  • ph.ecog: ECOG performance score (0=good 5=dead) # 病人ECOG表现得分
  • ph.karno: Karnofsky performance score (bad=0-good=100) rated by physician # 病人卡式得分
  • pat.karno: Karnofsky performance score as rated by patient # 病人卡式得分2
  • meal.cal: Calories consumed at meals # 每餐热量摄入
  • wt.loss: Weight loss in last six months # 6月内体重降低量

从age到wt.loss均可以作为研究影响肺癌病人生存时间的因素之一。

计算生存曲线：survfit()

将sex作为影响生存效应的自变量做生存分析：（time和status同时作为Kaplan-Meier survival estimate方法的因变量）；

  • $n$: total number of subjects in each curve. # 分类里所有样本数目
  • $time$: the time points on the curve. # 曲线的时间点
  • $n.risk$: the number of subjects at risk at time $t$. #曲线$t$时间点风险样本数目
  • $n.event$: the number of events that occurred at time $t$. # $t$时刻时间发生的数目
  • $n.censor$: the number of censored subjects, who exit the risk set, without an event, at time t.
  • $lower$,$upper$: lower and upper confidence limits for the curve, respectively. # 置信区间
  • $strata$: indicates stratification of curve estimation. If strata is not NULL, there are multiple curves in the result. The levels of strata (a factor) are the labels for the curves. # 曲线分层信息

生存曲线

使用survminer包里面的ggsurvplot函数去可视化生存曲线，它展示的信息有：

  • 95% confidence置信区间；
  • risk table 数目或者百分比的风险值；
  • Log-Rank test的P值；
  • median survival水平或垂直线；

The horizontal axis (x-axis) represents time in days, and the vertical axis (y-axis) shows the probability of surviving or the proportion of people surviving. The lines represent survival curves of the two groups. A vertical drop in the curves indicates an event. The vertical tick mark on the curves means that a patient was censored at this time.

  • At time zero, the survival probability is 1.0 (or 100% of the participants are alive).
  • At time 250, the probability of survival is approximately 0.55 (or 55%) for sex=1 and 0.75 (or 75%) for sex=2.
  • The median survival is approximately 270 days for sex=1 and 426 days for sex=2, suggesting a good survival for sex=2 compared to sex=1.

Log-Rank 检验比较组间生存曲线的差异：survdiff

Log-Rank检验用了比较组间生存曲线的是否存在显著差异，其零假设$H_0$是认为两组的生存曲线没有差异。该检验方法是非参数检验，根据两组数据事件的观测值与期望值之间的比较，发现组间是否存在差异。log rank统计量大概服从卡方检验分布。使用survival包的survdiff函数分析：

  • n: the number of subjects in each group.
  • obs: the weighted observed number of events in each group.
  • exp: the weighted expected number of events in each group.
  • chisq: the chisquare statistic for a test of equality.
  • strata: optionally, the number of subjects contained in each stratum.

复杂生存曲线

复杂生存曲线是指使用联合自变量去分析生存时间和生存事件的分析，如考虑sex，rx和adhere等

连续性自变量分类

在对基因等连续性自变量做生存曲线时，需要定义over-或者under-represented分组，一般处理方法是对齐进行Z-score化，然后根据1.96也就是95%的置信区间划分上述分组。

总结

Survival analysis is a set of statistical approaches for data analysis where the outcome variable of interest is time until an event occurs.

Survival data are generally described and modeled in terms of two related functions:

  • the survivor function representing the probability that an individual survives from the time of origin to some time beyond time t. It’s usually estimated by the Kaplan-Meier method. The logrank test may be used to test for differences between survival curves for groups, such as treatment arms.
  • The hazard function gives the instantaneous potential of having an event at a time, given survival up to that time. It is used primarily as a diagnostic tool or for specifying a mathematical model for survival analysis.
