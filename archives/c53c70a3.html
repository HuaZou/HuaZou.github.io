MaAsLin2：环境变量与微生物数据的关联分析R包

Posted on 2020-04-20 In R , DataAnalysis Views: Valine:
Symbols count in article: 3.6k Reading time ≈ 3 mins.

介绍

该R包目的是在大队列研究中找到微生物多组学特征和负责表型之间的关联关系。它是基于线性模型发现关联关系的软件，并且其中包含多种分析方法，归一化方法和数据转换等等。

原理

  • 归一化方法：TSS, CLR, CSS, TMM
  • 线性模型: LM, SLM, CPLM, ZICP, NEGBIN, ZINB
  • 数据转换方法：LOG, LOGIT, AST
  • 校正方法：BH, holm, hochberg, hommel, bonferroni, BY

通过转换数据，选择符合数据的模型，然后对其进行线性回归，最后校正回归显著性p值获取关联关系

输入数据

MaAsLin2 requires two input files.

  1. Data (or features) file # 微生物数据，行名-样本ID；列名-微生物特征
    • This file is tab-delimited.
    • Formatted with features as columns and samples as rows.
    • The transpose of this format is also okay.
    • Possible features in this file include taxonomy or genes.
  2. Metadata file # 表型数据 行名-样本ID；列名-理化指标等
    • This file is tab-delimited.
    • Formatted with features as columns and samples as rows.
    • The transpose of this format is also okay.
    • Possible metadata in this file include gender or age.

PS： R function模式，两类输入数据是data.frame格式

输出结果

MaAsLin2 generates two types of output files: data and visualization.（数据和可视化结果）

  1. Data output files
    • all_results.tsv # 所有关联关系结果以txt格式输出
      • This includes the same data as the data.frame returned.
      • This file contains all results ordered by increasing q-value. # 校正P值
      • The first columns are the metadata and feature names. # 第一列是特征名字
      • The next two columns are the value and coefficient from the model. # 检验值和相关系数
      • The next column is the standard deviation from the model. # 标准方差
      • The N column is the total number of data points.
      • The N.not.zero column is the total of non-zero data points.
      • The pvalue from the calculation is the second to last column.
      • The qvalue is computed with p.adjust with the correction method.
    • significant_results.tsv # 显著结果
      • This file is a subset of the results in the first file.
      • It only includes associations with q-values <= to the threshold.
    • residuals.rds # 每个特征的残差结果
      • This file contains a data frame with residuals for each feature.
    • maaslin2.log # 运行脚本的日志文件
      • This file contains all log information for the run.
      • It includes all settings, warnings, errors, and steps run.
  2. Visualization output files # 可视化输出结果
    • heatmap.pdf # 显著相关关系的热图结果
      • This file contains a heatmap of the significant associations.
    • [a-z/0-9]+.pdf # 每个显著相关关系的热图
      • A plot is generated for each significant association.
      • Scatter plots are used for continuous metadata.
      • Box plots are for categorical data.
      • Data points plotted are after normalization, filtering, and transform.
