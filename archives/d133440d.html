线性回归初认识

Posted on 2020-04-26 In Math , Statistics Views: Valine:
Symbols count in article: 2.3k Reading time ≈ 2 mins.

介绍

线性回归是利用数理统计中回归分析，来确定两种或两种以上变量间相互依赖的定量关系的一种统计分析方法，运用十分广泛。其表达形式为$y = w’x+e$，$e$为误差服从均值为0的正态分布。 回归分析中，只包括一个自变量和一个因变量，且二者的关系可用一条直线近似表示，这种回归分析称为一元线性回归分析。如果回归分析中包括两个或两个以上的自变量，且因变量和自变量之间是线性关系，则称为多元线性回归分析。

公式

  1. $y$ (response variable): 响应变量或因变量，通过解释变量或自变量(explanatory variables or independent variables) $x_{n}$预测；
  2. $\beta{n}$: 回归系数，决定每个解释变量对响应变量的解释程度（可以理解为在其他变量固定下，某个解释变量改变一单位则对应的响应变量改变$\beta{n}$单位）；
  3. $\varepsilon$: 残差，指所有解释变量预测得到的响应变量与真实响应变量之间的差值，该值反映了模型的拟合效果。

类别

  1. Simple linear regression：自变量和因变量均只有一个；

  2. Multiple linear regression：自变量不知一个，因变量仅仅一个；

  3. Logistic regression：因变量是二分类变量，回归曲线是$S$型；

  4. Ordinal regression：因变量是有序变量；

  5. Multinomial regression：因变量不是数值或者比例，仅仅是名义上的变量；

区别

1-2 类型的线性模型被称为general linear model, 而3-5 类型的线性模型一般归纳为generalized linear models，它与一般线性模型的最大区别是需要假定数据的分布情况。

For example, ordinal dependent variables can be modeled with cumulative link models. Binary (yes/no) dependent variables can be modeled with logistic regression. Dependent variables of discrete counted quantities can be modeled with Poisson regression and similar techniques. Percent and proportion data can be modeled with beta regression.

拟合方法

在拟合模式的过程中，一元一次的线性回归可以直接计算回归系数，而多元线性回归则使用OLS，广义线性模型使用MLE。

For any type of linear model, some method is used to find the value for the parameters for the model which best fit the data. For simple models like linear regression or a simple analysis of variance, these parameters estimates could be found by hand calculations. Luckily, more complex models can be fit by computer algorithm. General linear models are typically fit by ordinary least squares (OLS), whereas generalized linear models are typically fit by maximum likelihood estimation (MLE). OLS is actually a specific case of MLE that is valid only when the conditional distribution of the data are normal. These approaches to estimation are generally reliable, but the reader should be aware that there are cases where MLE may fail.
