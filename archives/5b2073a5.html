假设检验(hypothesis testing)

什么是假设检验

假设检验的逻辑是：在不确定性条件下进行决策选择；

假设检验存在两种情况：原假设即我们认为是正确的零假设（$H_0$）和与之相反的备择假设（$H_1$），这需要我们做出决定到底是相信$H_0$还是$H_1$；

假设检验过程中，不可能做出的结论总是正确的，必须要考虑误差的概率，假设检验中将可能发生错误分为两类：第一类错误，第二类错误

  • 第一类错误： 零假设$H_0$实际上是真的，但被假设验证为假的，犯了第一类错误
  • 第二类错误： 零假设$H_0$实际上是假的，但被假设验证为真的，犯了第二类错误

通常，将在假设检验是所指定的发生第一类错误的概率的最大容许值称为检验的显著水平，一般选择显著性水平为5%和1%。

假设检验步骤

  1. 数据筛选和离群值查看，使用boxplot图或者描述性统计量检查缺失数据和离群值，离群值的定义可以根据四分位数确定;

  2. 正态性检验：统计假设检验可分为参数检验和非参数检验。参数检验假定数据可以由已知的参数定义的分布很好地描述，而非参数检验是在总体方差未知或知道甚少的情况下，利用样本数据对总体分布形态等进行推断的方法。 由于非参数检验方法在推断过程中不涉及有关总体分布的参数，因而得名为“非参数”检验。一般使用QQ图（quantile）进行视觉评估;

  3. p值: 用于做出决定是否拒绝$H_0$的概率值.该值根据自身选择的分布计算得到的统计量确定的,并且在做多重假设检验时需要对所有假设检验的p值进行校正处理.

  4. 综上:

    ​ 假设检验的步骤

    1. 确定要检验的假设
    2. 选择检验统计量
    3. 确定拒绝域
    4. 求出检验统计量的p值
    5. 查看样本结果是否位于拒绝域
    6. 做出决策

错误类型

如前所述,假设检验中的错误类型分为两类:

  • I类错误: 假阳性; 用$\alpha$ 代替
  • II类错误: 假阴性; 用$\beta$代替
  • 检验效能: 统计检验的”效能”被定义为 $(1-\beta) * 100$
  • 效能分析:决定统计学检验的效能和计算揭示一定大小所需要的最小样本量
    • $\alpha$, I类错误;
    • $\beta$, II类错误;
    • $d$, 效应大小,即所研究的效应相对样本的标准差$\sigma$的大小;
    • $n$, 样本大小

灵敏度和特异度

  • 灵敏度: 被检测正确识别出来的阳性结果; 检测结果是真阳性/症状是阳性的比例;
  • 特异度: 被检测正确识别出来的阴性结果; 检测结果是真阴性/症状是阴性的比例;
  • 阳性预测值(PPV): 检测结果是阳性的病人中被正确诊断的比例; 真阳性/检测结果全是阳性的比例;
  • 阴性预测值(NPV): 检测结果是阴性的病人中被正确诊断的比例; 真阴性/检测结果全是阴性的比例.
  • 假阳性率 ($\alpha$), I类错误 = 1 - 特异度 = $\frac{FP}{FP+TN}$;
  • 假阴性率($\beta$), II类错误 = 1 - 灵敏度 = $\frac{FN}{FN+TP}$;
  • 效能 = 灵敏度 = 1 - $\beta$;
  • 阳性似然比 = ${sensitivity}{1-specificity}$;
  • 阴性似然比 = ${1 - sensitivity}{specificity}$.

受试者操作特征(receiver operating characteristic curve : ROC)曲线

ROC曲线是展示灵敏度(真阳性率)和特异度(假阳性率)之间关系的曲线,曲线下的面积为AUC((Area Under the Curve), AUC值常用来评估一个二值分类器的优劣.

均值检验方法

假设检验中的数值型数据的均值检验。比较不同组间的均值差异，根据随机样本的分布函数是否有参和无参再选择不同的检验方法。均值服从选择的检验分布，计算得到对应的统计量。一般情况下，有参检验服从正态分布并且有参检验要比非参数检验更灵敏，但是需要满足特定的假设。

非参检验方法

  • 符号检验或者配对秩和检验

  • U检验或者秩和检验

  • kruskal-Wallis 检验

  • 重复数据的friedman检验

有参检验方法

  • T 检验或者Z检验

  • 单因素方差分析

  • 多因素方差分析

总结

  • 参数检验和非参数检验的区别是前者已知总体分布参数，而后者未知，因此后者使用relative rank代替element value；
  • Kruskal-Wallis’ test 是非参数的单因素方差分析；
  • friedman test 是非参数的单因素重复数据的方差分析；
  • z检验适合大样本而T检验适合小样本（n=30）；
  • 一个标准正态分布，一个t分布，t分布更胖尾部更长，对极端异常值不敏感；
  • ANOVA方法和非参检验方法的区别不在随机数据是否符合正态分布，而是残差是否符合正态分布.

比率检验方法

  • 卡方检验；

  • Fisher检验；
