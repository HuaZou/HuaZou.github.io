A full workflow of miRNA-seq pipeline

miRNA可以调节靶基因的表达。miRNA序列较短，在测序过程中可能混入了其他RNA，因此miRNA的分析流程需要多次过滤非miRNA序列。另外在比对miRNA到reference时候，推荐使用短序列比对软bowtie。

MicroRNA (miRNA) 是一类内生的、长度约为20-24个核苷酸的小 RNA，其在细胞内具有多种重要的调节作用。每个 miRNA 可以有多个靶基因的表达，而几个 miRNA 也可以调节同一个基因的表达。据推测，miRNA 调节着人类三分之一的基因。

miRNA的特点

  1. hsa、mmu、rno分别代表人、小鼠、大鼠。

  2. mir、MIR、miR分别代表动物未成熟miRNA、植物未成熟miRNA、成熟 RNA。

  3. 即阿拉伯数字。代表miRNA发现的先后顺序。一般情况下，数字越小，发现越早。

  4. 对于相似度非常高但又不完全相同（如：仅差一两个碱基）的成熟miRNA，加上一个英文小写字母（a,b,c,…）以示区别。

install software

mirdeep2软件可以识别已知miRNA并鉴定出新miRNA，其涉及到的软件：

  • bowtie: 比对短序列
  • viennaRNA: 预测二级结构
  • SQUID library: 辅助randfold
  • randfold: 检测RNA的折叠自由是否属于miRNA

RNA data

  1. Rfam data
  2. Mus_ncRNA
  3. Musculus_UCSC exon.fa and intron.fa
  4. mirbase
  5. Mus_primary_assembly_dna

pipeline

Download reference and curate the data

该流程需要过滤掉non-coding RNA序列以及外显子内的类似miRNA序列，最后再将序列比对到mirbase的成熟和全部miRNA序列上，因此需要下载较多的参考数据。

  1. 下载ncRNA数据，ncRNA数据由musculus的ncRNA和Rfam数据库的RNA构成，合并两类fa后再进行建索引；

    step1: 从EBI的Rfam下载ncRNA数据，由于最新版14.2没有找到注释文件，暂时下载14.1版本（相较14.2版本，其少了10个ncRNA的序列）；

    step2: 下载release100的musculus ncRNA数据

    step3: 整合Rfam和Musculus的ncRNA数据，并获取rRNA、tRNA、sRNA和snRNA的fa文件以及使用makeblastdb建立其对应的索引，为后面过滤ncRNA服务。

    ​ 获取fa和建索引命令

  1. 下载UCSC的Musculus外显子和内含子数据（ENSEMBLE数据库需要自己提取该数据集，因此暂不使用）；

    ​ 上UCSC官网查找Musculus物种，然后按照教程下载外显子和内含子序列（Tools-> Table Browser）

  1. 因为mirdeep2软件只能识别没有空格的fa注释title，因此需要对Mus_primary_assembly_dna的title进行无空格处理；

    step1: 下载primary_dna.fa，并处理title的空格

Quality Scanning

Raw data质控前应该都需要对其进行扫描，检查其状况（测序长度、碱基质量、接头序列、GC分布等等），fastqc软件提供了最简单最易用的命令。 batch_fqc.pl

Generate fastqc shell

Remove adapter and filter low quality reads

Raw data通常包含测序的接头序列，因此可以使用cutadapt去除接头，再使用trimmomatic去除低质量的序列，后者设置参数的时候需要注意LEADING:20 TRAILING:20 SLIDINGWINDOW:4:20 MINLEN:17。batch_trim.pl

run the 2nd fastqc for trimmed data batch_fqc_2nd.pl

Generate trim shell

Convert fastq to fasta

在blast核酸比对中，需要使用fasta格式且还需要替换fasta的title，使其符合mirdeep2的fasta title格式。也即是如下格式：

物种_顺序标签_x其他

转换的perl脚本：batch_collapse.pl

Generate collapse shell

Remove ncRNA

miRNA测序过程可能包含了较多的非miRNA的ncRNA，比如rRNA, tRAN, sRNA和snRNA等，因此需要将其剔除掉，batch_blast.pl

提取unannotated fasta by using get_unannotated_fasta.py

Generate blast shell

Remove miRNA in exon

大部分miRNA是位于内含子区域，但上述得到的序列可能存在外显子区域，因此将上一步过滤得到的fasta文件先比对到exon参考序列，如果无法比对上则直接保留，否则继续比对到intro序列，比对上则保留，无法比对上则丢弃。batch_mapped_intron.pl

generate mapping shell

Discover known and novel miRNA
