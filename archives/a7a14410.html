机器学习（六）：调参

Posted on 2021-01-16 In Course , Machine learning Views: Valine:
Symbols count in article: 9k Reading time ≈ 8 mins.

介绍

调参是选择最佳模型的必需过程，在调参前，我们需要明确什么情况下是最佳模型，评估指标是什么，哪些因素影响评估指标。调参有如下选择：

  • 什么参数可以调整

  • 什么方法适合查找最适合模型的参数

  • 使用什么测试选择限制模型过拟合训练集

Dataset

随机算法

模型使用randomforest算法，对mtry和ntree参数进行调参。

  • mtry: 训练的随机markers数目

  • ntree: 决策树数目

评估模型效能的参数：

  • 评估分类器模型的参数是Accuracy 和 kappa

  • 评估回归模型的参数是RMSE, $R^2$和mean absolute error(MAE)

简单应用

调参(Caret)

Caret根据参数对模型效能的影响，设置了部分参数可调，比如随机森林中的 mtry，ntree参数，前者可调后者则较难。另外，调参的方式也有多种：

  • 随机搜索(Random Search)

  • 网格搜索(Grid Search)

随机搜索(Random Search)

在给定参数的范围内，随机搜索组合调节参数。它适合于无先验知识的调参即事先不知道哪些参数对模型效能提升最大。该方法能有效避免主观的bias。

Notes: 选择Accuracy最大值时候的mtry也即是markers数目

网格搜索(Grid Search)

每个参数列表均为网格内参数的线性组合

调参工具（Tune Using Algorithm Tools）

randomForest包提供了tuneRF函数，可设置mtr参数

设置参数搜索方法

如果需要根据数据集定制不同的必须训练参数，则有如下通用方法：

  • 手动调参(Tune Manually): 创建多个基于不同参数的模型并比较它们的优劣性

  • 扩展Caret(Extend Caret): 扩展算法的额外参数

手动调参(Tune Manually)

Notes: 根据不同ntree的模型Accuracy可以设置模型训练时的ntree数

扩展Caret(Extend Caret)

自定义caret的method，方便根据算法设置参数。

总结

模型调参需要对模型的算法了解，知道其有何种参数。在了解参数后，可是设置一系列参数范围，然后选择随机或网格方法选择参数用于训练，当然还有另外手动调参或重新撰写算法参数用于caret包，这适用于无法直接通过train等函数设置的算法参数。
