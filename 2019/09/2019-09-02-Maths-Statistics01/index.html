统计学基础知识

发表于 2019-09-02 分类于 数学学习 Valine：

概念

统计学是在资料分析的基础上，研究测定、收集、整理、归纳和分析反映数据资料，以便给出正确讯息的科学。这一门学科自17世纪中叶产生并逐步发展起来，它广泛地应用在各门学科，从自然科学、社会科学到人文学科，甚至被用于工商业及政府的情报决策。随着大数据（Big Data）时代来临，统计的面貌也逐渐改变，与资讯、计算等领域密切集合，是资料科学（Data Science）中的重要主轴之一。

1.1 总体和样本

总体： 包括数据集中所有元素

样本： 由总体中的一个或者多个元素组成的集合，一个总体可以构成多个样本

参数：总体的特征，参数可以是均值或者标准差等描述性统计指标

统计量：一个样本的可测量的特征。如

  • 样本的均值；
  • 样本的极差；
  • 样本的方差等；

抽样分布：基于随机样本的给定统计量的概率分布

统计推断：根据从总体抽样的样本的统计量推断总体水平的特征的方法

1.2 概率分布

离散分布：投掷硬币；

连续分布：身高分布，可以使用每个元素出现概率作为y轴概率，然后构成概率密度函数（PDF）

期望值：是总体的观测值的概率分布函数的平均值，而样本均值是样本的观测值的平均值

方差：数据偏离期望值的程度，描述样本数据的波动情况。

自由度：自由度反应的样本的数目

1.3 单变量分布的描述

均值 ：算术平均值；

中位数：中位数是当数据按照顺序排列时中间的值；

众数：出现频率最多的数值

几何均值：通过计算每个数值的对数的算术平均值得到，表示分布的位置?

1.4 量化变异度

极差： 极大值和极小值的差值

百分位数： 累积分布函数（CDF）计算

标准差和方差： 数据与均值的离异程度

标准误： 是系数标准误的估计，对均值的估计值

置信区间：参数的真是范围区域

1.5 分布形状的参数描述

位置：由均值确定，连续分布函数的概率密度函数在x轴上的位置

尺度：概率分布函数的宽度

形状参数：偏度和峰度，如果分布不均匀则是偏斜分布，峰度衡量概率分布的”陡峭程度“
