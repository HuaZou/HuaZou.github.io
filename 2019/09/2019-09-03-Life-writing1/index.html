学位论文撰写（一）：如何避免口语化的句子

发表于 2019-09-03 分类于 生活笔记 阅读次数： Valine：
本文字数： 658 阅读时长 ≈ 1 分钟

前提

学位论文写作完成并且提交给技术导师审核，但返回来的意见则是文章是一篇口语化的论文。我对这样的结果一点都不感到惊讶。从我开始写论文到完成，我在这个过程一直感觉自己的写作能力几乎为零，很多句子我没有办法让自己满意，但为了赶进度和凑够字数，很多时候我都是想到什么就写什么，完全没有考虑到怎么用书面形式的学术术语去表达。这样的原因我认为主要有一下几点：

  • 自身学术修养和写作能力的局限性；
  • SCI论文迟迟没有接收，导致延毕发生并且现阶段论文还没有任何消息，自己内心已经接近崩溃；
  • 因为SCI论文的不确定性，毕业时间也就不确定，因此根本没有用心在写作上，只想着敷衍过去；
  • 除了这些因素外，还有在组里没有任何项目可做，自身价值得不到体现，感觉一事无成，偶尔能做的事情也就是打杂或做些对我自身科研能力没有任何提升的分析工作。

多种内外原因交织在一起，导致学位论文的质量如此低下。客观原因如环境因素是暂时无法解决的，因此提升自身学术修养和写作能力才是最关键的。为了我查阅了很多资料，得出的结论是看得文章太少、没有积累足够多的专业学术用语。

做法

  1. 多看文献，毕业文章是中文撰写，可以上知网、万方等网址下载相关领域的中文文献阅读，仔细琢磨文章的句子；
  2. 多使用学术用语，不能使用常用的大白话词语，在需要使用学术名词的地方一定要使用学术名词；
  3. 描述科学问题不能追求定性描述，而应该尽量定量描述。这也是学术论文和散文的最大区别；
  4. 对研究的问题认识不够，需要从知网下载相关中文文献。
