Rmarkdown: 一个科学编写数据分析的框架

发表于 2019-09-30 更新于 2019-09-20 分类于 编程入门 Valine：

介绍1

R markdown文件整合了R代码和markdown的特性，这使得R代码、R结果和markdown的注释信息结合在一起方便分析人员沟通。可以使用Rstudio软件打开R markdown文件，在运行前需要安装rmarkdown包。

R markdown可以生成多种报告文件，因此含有了大量的规则。详细规则参考官网给出的rmarkdown cheatsheet2以及rmarkdown reference3。使用步骤如下几步：

  1. 新建Rmd文件；
  2. 编写Rmd文件；
  3. 渲染Rmd文件；
  4. 预览渲染结果；
  5. 上传公开结果；
  6. 检查渲染日志；
  7. 使用输出文件。

工作原理4

运行knitr后，1）Rmd文件先执行所有代码块，生成具有结果在内的新的markdown（.md）文件；2）markdown文件通过pandoc软件会生成各种格式的结果文件（）。Web page, PDF, MS Word document, slide show, notebook, handout, book, dashboard, package vignette 或者 other format。使用render函数运行上述步骤或者直接点击Rstudio内的knitr按钮。

代码块5

代码块内的代码可以是R、python和bash等；代码块输出选择参数也可以设置；另外，代码块的全局变量通过knitr::opts_chunk$set设置。

  • include = FALSE prevents code and results from appearing in the finished file. R Markdown still runs the code in the chunk, and the results can be used by other chunks.
  • echo = FALSE prevents code, but not the results from appearing in the finished file. This is a useful way to embed figures.
  • message = FALSE prevents messages that are generated by code from appearing in the finished file.
  • warning = FALSE prevents warnings that are generated by code from appearing in the finished.
  • fig.cap = "..." adds a caption to graphical results.
