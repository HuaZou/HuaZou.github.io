markdown：轻量级标记语言

什么是markdown

markdown是一种方便编辑纯文本格式文档的轻量级标记语言，其文档通常以md为后缀，并且文档可以多种格式文件类型导出，包括Word，HTML和PDF等格式。该语言被广泛应用许多网站，如大家熟悉的GitHub，简书等。

编辑器

本人推荐使用编辑器Typora，该软件可跨window，Linux和Mac os等平台，并且其菜单栏的段落选项包含几乎所有的规则。

语法规则

markdown语言的语法规则比较简单。主要有标题、列表、段落区块、代码、连接和图片以及数学公式等。

标题

通过#表示1-6级标题，一级标题对应一个 # 号，二级标题对应两个 # 号，以此类推。

段落(正文)

Markdown 段落没有特殊的格式，直接编写文字就好，段落的换行是使用两个以上空格加上回车或者使用空行。

  • 字体：正文内字体
  • 分割线：在一行中用三个以上的星号、减号、底线来建立一个分隔线，行内不能有其他东西。你也可以在星号或是减号中间插入空格。
  • 删除线：正文内突出某些需要删除的内容。
  • 下划线：使用html语言的<u>下划线</u>表示下划线。
  • 脚注：对正文内容的引用说明等。

列表

支持有序列表和无序列表以及嵌套列表。无序列表使用星号(*)、加号(+)或是减号(-)作为列表标记。有序列表使用数字并加上**.**号来表示。

区块

区块引用是在段落开头使用 > 符号 ，然后后面紧跟一个空格符号。

代码块

可以用 ``` 包裹一段代码，并指定一种语言（也可以不指定）。

链接

普通链接和高级连接

图片

  • 开头一个感叹号 !
  • 接着一个方括号，里面放上图片的替代文字
  • 接着一个普通括号，里面放上图片的网址，最后还可以用引号包住并加上选择性的 ‘title’ 属性的文字。

表

表格使用竖线来分隔不同的单元格，使用-来分隔表头和其他行。

  • -: 设置内容和标题栏居右对齐。
  • :- 设置内容和标题栏居左对齐。
  • :-: 设置内容和标题栏居中对齐。

有向图

在代码块选择sequence、flow等语言画流程图 。

数学公式

可以使用两个美元符 $$ 包裹 TeX 或 LaTeX 格式的数学公式来实现。提交后，问答和文章页会根据需要加载 Mathjax 对数学公式进行渲染。
